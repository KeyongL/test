# ⚡ 快速测试指南

## 1分钟快速验证

### 启动应用
```bash
streamlit run app.py
```

### 测试流程

1. **填写问卷**
   - 在左侧选择 "📝 填写问卷"
   - 快速回答前3题（测试用）
   - 点击"提交"

2. **查看数据**
   - 左侧切换到 "📊 查看数据"
   - 输入密码：`admin123`
   - 应该能看到刚才提交的数据

3. **验证数据库**
   - 检查项目目录，应该看到 `survey_data.db` 文件
   - 这个文件就是你的数据库

---

## 预期结果

✅ 问卷可以正常填写和提交  
✅ 数据保存到 `survey_data.db`  
✅ 查看页面能显示数据  
✅ 可以导出 CSV  

---

## 如果遇到问题

**问题**：页面空白或报错  
**解决**：检查是否安装了所有依赖 `pip install -r requirements.txt`

**问题**：数据库无法创建  
**解决**：检查项目目录的写入权限

**问题**：查看数据页面空白  
**解决**：确保先提交至少一份问卷
