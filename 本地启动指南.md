# 🚀 本地启动指南

## 第一阶段：本地运行测试

### 步骤 1：检查环境

确保已安装 Python 3.7+：
```bash
python --version
```

### 步骤 2：安装依赖

```bash
pip install -r requirements.txt
```

### 步骤 3：启动应用

```bash
streamlit run app.py
```

### 步骤 4：访问应用

浏览器会自动打开，或手动访问：`http://localhost:8501`

---

## 功能说明

### 📝 填写问卷页面
- 左侧选择 "📝 填写问卷"
- 逐题回答 13 个问题
- 提交后数据自动保存到 `survey_data.db` 数据库

### 📊 查看数据页面
- 左侧选择 "📊 查看数据"
- 输入密码：`admin123`（可在代码中修改）
- 查看所有提交的数据
- 导出 CSV 文件
- 查看简单的统计图表

---

## 数据存储

- **数据库文件**：`survey_data.db`（SQLite 格式）
- **位置**：项目根目录
- **格式**：所有答案以 JSON 格式存储，更灵活易扩展

## 常见问题

**Q: 数据库文件在哪里？**
A: 项目根目录下的 `survey_data.db`，首次运行会自动创建。

**Q: 如何修改查看数据的密码？**
A: 在 `app.py` 中找到 `if password != "admin123":` 这行，修改 `"admin123"` 为你想要的密码。

**Q: 如何清空数据重新开始？**
A: 删除 `survey_data.db` 文件，下次运行会自动创建新数据库。
