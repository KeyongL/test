# 🤖 可复用问卷系统模板

基于 Streamlit 的可复用问卷调查系统，支持快速配置和部署。

## ⚡ 快速开始

**3步完成新问卷部署**：

1. **修改配置**：编辑 `survey_config.json`（3分钟）
2. **本地测试**（可选）：`streamlit run app.py`
3. **部署上线**：`git push`（自动部署）

**详细使用指南**：👉 [快速复用指南.md](快速复用指南.md)

---

## 📋 当前项目：AI智能体赋能教学调研

专为高校教师教学效率调研设计的问卷系统。

## 🚀 功能特点

- **分页式问卷**：一次只显示一个问题，提升用户体验
- **智能数据收集**：自动保存所有回答到 CSV 文件
- **简洁界面**：Google 风格设计，清晰易用
- **多题型支持**：单选题和多选题混合
- **数据持久化**：本地 CSV 文件存储，便于后续分析

## 📋 调研内容

包含 13 个核心问题，涵盖：
- 基础画像（教学/科研重心）
- AI 工具使用情况
- 教学效率提升需求
- 技术接受度和培训需求

## 🛠️ 安装运行

### 环境要求
- Python 3.7+
- Streamlit
- Pandas

### 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 运行应用
streamlit run app.py
```

### 访问应用
应用启动后，在浏览器中访问：http://localhost:8501

## 📊 数据管理

- 所有调研数据自动保存到 `survey_data.csv`
- 文件包含时间戳和所有回答
- 支持后续数据分析和导出

## 🔧 项目结构

```
survey-app/
├── app.py              # 主应用文件
├── requirements.txt    # Python 依赖
├── survey_data.csv     # 数据存储文件
└── README.md          # 项目说明
```

## 📝 使用说明

1. **开始调研**：访问应用主页
2. **回答问题**：逐题完成问卷
3. **提交数据**：最后一题点击"提交"
4. **数据收集**：所有回答自动保存

## 📈 数据分析

调研数据包含以下维度：
- 用户角色和职责分布
- AI 工具熟悉程度
- 教学痛点识别
- 培训需求和偏好
- 技术接受度评估

## 🔒 隐私说明

- 不收集个人身份信息
- 所有数据本地存储
- 仅用于教学研究目的
